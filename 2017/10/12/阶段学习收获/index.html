码云pages是2016年出来的。

在这里我简单的说一下github的使用

========================================分割线===================================================

##一。创建一个新的项目##

码云不同于github,他可以在远程建文件，所以你必须把他的.git文件clone下来


直接点创建就好，别忘了你的项目名字！！！

##二。克隆到你需要上传的文件夹里面 ##

点击右边的黄色框框克隆！！！！然后如下图：

别忘了使用git命令

克隆之后你会发现你的文件夹里多了一个包，那个包名就是你的项目名称，且.git文件就在里面

将一整个文件夹里面的文件全部移到这个包里。如下图：

接下来你需要上传到项目上去了

在这个包里面打开命令行 输入：

这三条命令

git add .
git commit -m ""
git push -u origin master

上传成功后，刷新你的项目页面。就可以看到上传上去的文件了

##三。启动pages服务##

注意，pages服务只会读取根目录下的index.html

然后你直接点击启动就可以了，就会出现如下页面：

上面的连接就是你的index.html的内容了
