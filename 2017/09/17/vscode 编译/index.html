vscode 编译

  • 按 ctrl+‘~’号键
  • 输入 cd 用来进入到正在写的文件里
  • 输入 cd .\main 文件不多时，输入文件首字母后按Tab 键系统会自动补齐
  • 输入 ls
  • 输入 node .\mian.js 同样用Tab 键来补齐全名

快捷键

  • ctrl +Z 撤销
  • ctrl +X 剪切
  • ctrl +C 复制
  • ctrl +V 黏贴
  • ctrl +A 全选
  • ctrl +/ 注释
  • windows + R 命令行
  • shift + 鼠标右键 菜单会出现：在此处打开命令行窗口
